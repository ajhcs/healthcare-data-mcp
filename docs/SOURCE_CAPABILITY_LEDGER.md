# Source Capability Ledger

| User claim | Exact source | Tool | Status | Caveat | Unsupported substitutions |
|---|---|---|---|---|---|
| ACGME program IDs | ACGME public Program Search export / data dictionary | `workforce_analytics.get_acgme_source_status`, `get_acgme_program`, `search_acgme_programs` | `ready` or `import_required` | Import-backed unless ACGME publishes a stable documented unauthenticated API. | Claiming complete live inventory without a ready import |
| ClinicalTrials.gov sponsor inventory | ClinicalTrials.gov v2 studies using `query.spons` | `research_trials.inventory_clinical_trial_sponsors` | `ready` API scan | Sponsor roles are counted separately; aliases are explicit and reviewed. | Raw sponsor search as deduped entity inventory |
| ClinicalTrials.gov site inventory | ClinicalTrials.gov v2 study locations using `query.locn` | `research_trials.inventory_clinical_trial_sites` | `ready` API scan | Site key is facility plus city/state/country/ZIP; blank facilities are unresolved. | Facility-name-only aggregation across cities/states |
| CMS cybersecurity attestation | Reviewed public source with explicit attestation field | `public_records.get_cyber_attestation_source_status` | `not_publicly_available` | No configured public CMS source supports broad cybersecurity attestation status. | PI/EHR rows, OCR breaches, SEC cyber disclosures, CISA KEV |
| OCR breaches | HHS OCR breach portal large breach rows | `public_records.get_breach_history` | import/acquire-backed | Supports breach facts only for rows loaded from OCR. | Broad security posture or attestation |
| State AG notices | Reviewed state source rows imported locally | `public_records.get_state_ag_breach_notice_sources`, `search_state_breach_notices` | state-specific `ready`, `not_searchable`, `not_automatable`, or `import_required` | Not a national cache; every indexed notice needs a source URL/file. | Inferring notice absence from missing import |
| AMI 30-day mortality | CMS Provider Data Catalog `MORT_30_AMI` / `MORT-30-AMI` | `hospital_quality.get_quality_measure_rows` | exact row only | Source-shape errors are returned if CMS changes the file shape. | PHC4 in-hospital mortality |
| Hospital-wide readmission | CMS Provider Data Catalog `READM_30_HOSP_WIDE` / `READM-30-HOSP-WIDE` | `hospital_quality.get_quality_measure_rows` | exact row only | HRRP condition rows remain available through `get_readmission_data` as adjacent context. | HRRP AMI/HF/PN/COPD/HIP-KNEE/CABG condition rows |
| CLABSI SIR | CMS Healthcare-Associated Infections `HAI_1_SIR` / `HAI-1` | `hospital_quality.get_quality_measure_rows` | exact row only | HAC Reduction Program totals remain available through `get_safety_scores` as adjacent context. | HAC total score or null HAC domains |

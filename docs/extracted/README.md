not active server implementations yet.

- `mr-explore/`: MCP contract, local rate-data query core, hospital

  prototype code from `MR-Explore/`.

## Promotion Rule

Before using extracted material in production MCP servers, adapt it into the
owning server/package, add tests, and record the source path in the PR.
